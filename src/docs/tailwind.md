# Tailwind CSS v4

## Основная идея

Tailwind — это не библиотека компонентов, а набор утилит.

Интерфейс строится путём комбинирования небольших классов.

---

## Почему Tailwind

- высокая скорость разработки;
- единый стиль;
- меньше собственного CSS;
- отсутствие конфликтов имён классов.

---

## Контейнер

```html
class="mx-auto w-full max-w-7xl px-6 lg:px-8"
```

Разбор:

| Класс       | Значение                               |
| ----------- | -------------------------------------- |
| `mx-auto`   | центрирует блок                        |
| `w-full`    | ширина 100%                            |
| `max-w-7xl` | максимальная ширина                    |
| `px-6`      | горизонтальные отступы                 |
| `lg:px-8`   | увеличенные отступы на больших экранах |

---

## Адаптивность

Префиксы:

- `sm:`
- `md:`
- `lg:`
- `xl:`
- `2xl:`

Они соответствуют брейкпоинтам Tailwind.

При необходимости можно определить собственные.
