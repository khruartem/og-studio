# Архитектура проекта

## Цель

Создать производительный и масштабируемый сайт цифровой студии на Astro.

Проект должен быть удобен для сопровождения и расширения, а также служить примером современной фронтенд-архитектуры.

---

## Технологический стек

- Astro
- React
- TypeScript
- Tailwind CSS v4
- Vite

---

## Основные принципы

### Astro First

Все статические компоненты реализуются на Astro.

React используется только там, где необходима интерактивность.

---

### Composition over Duplication

Повторяющиеся элементы выносятся в переиспользуемые компоненты.

---

### Design System

Проект строится вокруг единой системы:

- цветов;
- типографики;
- размеров;
- компонентов.

---

### Минимум JavaScript

На клиент отправляется только тот JavaScript, который действительно необходим.
