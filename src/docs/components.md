# Компоненты Astro

## Container

### Назначение

Ограничивает максимальную ширину контента и задаёт горизонтальные отступы.

Пример:

```astro
<Container>
  <slot />
</Container>
```

Используется практически во всех секциях сайта.

---

## Section

### Назначение

Семантическая обёртка над `<section>`.

Отвечает только за:

- семантику;
- использование `Container`.

Не отвечает за:

- размеры отступов;
- фон;
- оформление.

Такие вещи определяются непосредственно в месте использования через `class`.

Пример:

```astro
<Section
  id="services"
  class="pt-24 pb-20 bg-surface"
/>
```

---

## Slot

`<slot />` — место, куда вставляется содержимое компонента.

Напоминает `children` в React, но является частью языка Astro.
